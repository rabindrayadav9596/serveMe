# serveme-api

## How to run

 > cd src/
  
 > python -m venv api
  

 > MacOS: source env/bin/activate
 > Windows: api\Scripts\Activate

 > MacOS: export FLASK_APP=serveme
 > Windows: set FLASK_APP=serveme

 > MacOS: export FLASK_DEBUG=1
 > Windows: set FLASK_DEBUG=1

 > pip install -r requirements.txt

 > flask run
  
  
 
  
 > MacOS: python run.py
